# Contributing to AI Debate Lab

Contributions are welcome! Here's how to get started.

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork and create a branch (`git checkout -b feature/your-idea`)
3. **Make changes** and test locally
4. **Commit** with clear messages (`git commit -m "Add feature: ..."`)
5. **Push** to your fork and open a **Pull Request**

## Running the App

```bash
pip install -r requirements-streamlit.txt
cp .env.example .env
# Add your OPENROUTER_API_KEY to .env
streamlit run app.py
```

## Code Style

- Use Python 3.10+ features where helpful
- Keep the Streamlit app responsive and the debate flow clear

## Questions?

Open an [issue](https://github.com/MrFunnything99/AI-Debate-Team/issues) and we'll help out.
